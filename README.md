# Agent Constraints

Agent Constraints defines reusable policy constraints for AGenNext agents, teams, frameworks, and products.

## Scope

This repository owns reusable constraints such as:

- LLM provider selection constraints
- tool adoption constraints
- license constraints
- security constraints
- cost/billing constraints
- data handling constraints
- human approval constraints

## Consumers

- Agent-Team
- Agent-Frameworks
- Agent-Knowledge
- future AGenNext products

## Principle

```text
Policies live here.
Implementations consume them.
```
